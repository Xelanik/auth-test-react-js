# React + TypeScript + Vite

This project provides a minimal functionality for auth and validation without using any libraries like `react-hooks-form`.

Used libs:

- `Tailwind`
- `Axios`
- `react-toastify`

## Local installation and usage

- `npm install`
- `npm run dev`

## Deployed to Netlify

- [React Auth App](https://shimmering-crepe-80a8b2.netlify.app)
